## 3. 未覆盖需求清单

### 3.1 关键缺失（影响演示）

| 需求 | 模块 | 说明 |
|------|------|------|
| 无登录页 | 2.2 | 所有页面直接访问，无权限隔离 |
| 盘笼移位登记 | 4.3 | 无移位 Dialog 和历史记录 |
| 真实文件导出 | 4.1 / 5.1 | 仅 Toast，无文件生成 |
| 权限树 Switch 保存 | 3.2 | Switch 修改后无保存逻辑 |

### 3.2 架构级缺失（生产环境必需）

| 需求 | 说明 |
|------|------|
| WebSocket 实时推送 | 站点状态、任务进度等无实时更新 |
| 数据库持久化 | 页面刷新后数据恢复初始状态 |
| ADFS/LDAP 集成 | 无登录认证 |
| 邮件发送 | 测试邮件仅 Toast |
| 后台任务 | 导出等操作无异步执行 |
