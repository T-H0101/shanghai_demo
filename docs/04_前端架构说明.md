### 3.2 核心组件说明

#### 3.2.1 AppShell（`components/layout/app-shell.tsx`）

**职责**：所有页面的统一外壳，提供侧边栏和顶栏

```typescript
export function AppShell({ children }: { children: ReactNode }) {
  const [sidebarOpen, setSidebarOpen] = useState(false)
  return (
    <div className="min-h-screen bg-slate-100">
      <Sidebar isOpen={sidebarOpen} onClose={() => setSidebarOpen(false)} />
      <div className="lg:ml-60">
        <Header onMenuClick={() => setSidebarOpen(true)} />
        <main className="p-4 lg:p-6 space-y-4 lg:space-y-6">
          {children}
        </main>
      </div>
    </div>
  )
}
```

#### 3.2.2 Header（`components/dashboard/header.tsx`）

**职责**：顶栏，包含全局搜索、状态信息、通知中心、用户信息

**关键交互**：
- 通知图标点击 → 展开通知 DropdownMenu
- 移动端菜单按钮 → 触发侧边栏展开

#### 3.2.3 Sidebar（`components/dashboard/sidebar.tsx`）

**职责**：左侧导航栏，包含 Logo、导航菜单

**导航项**：
- 首页仪表盘（`/`）
- 站点管理（`/sites`）
- 盘架管理（`/racks`）
- 任务管理（`/tasks`）
- 用户与权限（`/users`）
- 审计日志（`/logs`）
- 统一检索（`/search`）
- 系统设置（`/settings`）

#### 3.2.4 DetailPanel（`components/platform/detail-panel.tsx`）

**职责**：详情面板组件，用于列表页右侧详情展示

**Props**：
- `title`：面板标题
- `subtitle`：副标题/描述
- `empty`：是否显示空状态
- `actions`：操作按钮区域

#### 3.2.5 StatCard（`components/platform/stat-card.tsx`）

**职责**：统计卡片，用于展示关键指标

**Props**：
- `title`：卡片标题
- `value`：数值
- `unit`：单位（可选）
- `icon`：图标组件
- `iconBg/iconColor`：图标颜色配置
- `badge`：右上角标签
- `footer`：底部附加信息

## 4. store/notification 说明

### 4.1 架构设计

通知中心使用**模块级单例 + useEffect 订阅模式**，实现跨组件状态共享：

```
store/notification.ts（模块级）
├── notifState（状态对象）
├── listeners（订阅者 Set）
├── notifyListeners()（通知所有订阅者）
└── useNotificationStore()（React Hook）
```

### 4.2 状态结构

```typescript
interface Notification {
  id: string
  title: string
  message: string
  time: string
  type: 'info' | 'warning' | 'error' | 'success'
  read: boolean
}

interface NotificationState {
  notifications: Notification[]
  unreadCount: number
}
```


### 4.4 局限性

- 页面刷新后恢复初始状态（内存状态不持久化）
- 不支持多标签页同步状态
- 未使用 WebSocket，适合 Demo 阶段

## 6. Provider / Toast / Layout 说明

### 6.1 Providers（`components/providers.tsx`）

```typescript
export function Providers({ children }: { children: React.ReactNode }) {
  return (
    <>
      {children}
      <Toaster />
    </>
  )
}
```

**作用**：包裹在 `RootLayout` 中，为整个应用提供 Toast 上下文

### 6.2 Toast 系统

```
hooks/use-toast.ts
├── toast() 函数（触发 Toast）
├── useToast() Hook（获取 state）
├── reducer（状态管理）
└── listeners（订阅者列表）

components/ui/toast.tsx
├── ToastProvider
├── ToastViewport
├── Toast
├── ToastTitle
├── ToastDescription
├── ToastClose
└── ToastAction

components/ui/toaster.tsx
└── Toaster（最终渲染组件）
```

**使用方式**：
```typescript
import { toast } from "@/hooks/use-toast"

toast({ title: "操作成功", description: "数据已保存" })
toast({ title: "操作失败", variant: "destructive" })
```

### 6.3 Layout 系统

```
app/layout.tsx（RootLayout）
├── <html lang="zh-CN">
├── <Providers>
│   └── <Toaster />
└── <body>

components/layout/app-shell.tsx（AppShell）
├── <Sidebar />
├── <Header />
│   ├── <通知中心 />
│   └── <用户信息>
└── <main>{children}</main>
```
