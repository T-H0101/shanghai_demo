## 一、项目概述

### 1.1 核心功能

将待提交的档案数据自动批量打包为 ISO 镜像文件，方便数据移交与长期保存。

### 1.2 平台要求

跨平台支持：Linux、Windows、macOS

### 1.3 系统定位

新系统作为管理层，架构在原有光盘库系统之上，专注于设备管理和任务管理。

| 任务进行状态 | 进行中、已完成等 | - |
| 操作 | 导出、取消等 | - |

### 4.3 封包任务进度展示

#### 4.3.1 可视化图形查看

圆形节点可视化，动态展示加载效果：

**整体流程节点：**
```
开始 → 下载 → 等待 → 准备 → 分盘 → [多线程并行] → 结束
```

**准备节点显示内容：**
- 归档文件数量
- 总文件大小

**多线程并行展示：**
```
线程1: 等待 → 空节点 → 封包(进度%/分包速度) → 空节点 → 结束
线程2: 等待 → 空节点 → 封包(进度%/分包速度) → 空节点 → 结束
线程3: 等待 → 空节点 → 封包(进度%/分包速度) → 空节点 → 结束
...
```

所有线程结束后，统一执行到结束节点。

#### 4.3.2 线性加载条查看

- 展示线性加载条动画
- 与图形化流程对应
- 做好加载动画效果

### 4.4 常驻显示数据

每个任务下方存在常驻显示数据（允许收起）：

| 数据项 | 说明 |
|--------|------|
| 扫描文件数量 | 总文件数量 |
| 总文件大小 | 源数据总体积 |
| 总封包数量 | 已生成数据包数 |
| 成功数 | 成功完成数 |
| 异常数 | 异常/失败数 |

## 五、封包统计模块

### 5.1 查询逻辑

| 查询维度 | 说明 |
|---------|------|
| 任务 | 按任务筛选 |
| 子任务 | 按子任务筛选 |
| 状态 | 按封包状态筛选 |

### 5.2 封包统计列表字段

| 字段 | 说明 | 格式 |
|------|------|------|
| 任务名称 | 所属任务名称 | - |
| 子任务名称 | 所属子任务名称 | - |
| 包名称 | 打包后的文件名 | 如：433030-001.iso |
| 包大小 | 数据包体积 | - |
| 文件数量 | 包内文件数量 | - |
| 开始时间 | 封包开始时间 | yyyy-mm-dd hh:mm:ss |
| 完成时间 | 封包完成时间 | yyyy-mm-dd hh:mm:ss |
| 封包状态 | 进行中 / 已完成 等 | - |
| 存正情况 | 封包中显示 `-`，完成显示 `已存正` | - |
| 操作 | 预览、下载、数据明细 等 | - |

### 5.3 包数据明细

| 字段 | 说明 |
|------|------|
| 序号 | 文件编号 |
| 文件名称 | 包内文件名称 |
| 文件大小 | 文件体积 |
| 校验码 | 校验算法（如 SM3） |
| 文件路径 | 源路径保存到哪 |
| 存证情况 | 存证状态 |
| 操作 | 预览、下载电子存证报告 |

### 5.4 封包明细导出

| 配置项 | 说明 |
|--------|------|
| 选择任务 | 必选，父级筛选 |
| 选择子任务 | 可选，级联选择（无任务时不可选） |
| 导出内容 | 包含封包路径 |

### 5.5 封包文件导出

支持将封包文件整体导出。
