## 一、项目定位

**本系统是集团层统一管控平台，不替代各站点原有系统。**

核心能力：
- 数据同步（汇聚多站点数据）
- 统一视图（跨站点汇总展示）
- 统一权限（账号权限分配）
- 统一任务管理（跨站点任务调度）
- 日志管理（审计日志）
