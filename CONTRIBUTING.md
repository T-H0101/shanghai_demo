# Contributing to Optical Disc Library Management Platform

Thank you for your interest in contributing!

## Project Type

This is a **frontend demo project** with Mock data. It is **not a production system**.

- No real ADFS / LDAP / OAuth / SAML authentication
- No real database or backend API
- No real data synchronization with physical disc libraries

## How to Run

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Lint check
npm run lint
```
